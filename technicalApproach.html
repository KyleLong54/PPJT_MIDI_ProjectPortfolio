Peter Pan Junior Theater, or PPJT, is a community theater group that operates out of San Diego, California. In middle and high school, I was involved in their productions, and my family still works with them to this day.

What is MIDI?

MIDI, or Musical Instrument Digital Interface, is a format for musical data to be stored. Rather than storing the actual sound of the music, it stores the timing of events in the song. For example, turning a note on or stopping it is a MIDI event. This means that songs are made up of events that start and stop the various notes of the song.

Why make a MIDI application?

Peter Pan Junior Theater, during their rehearsals for their upcoming productions, uses a MIDI sequencing application called "Cakewalk Express". This application was last updated in the 1990s, and only runs on Windows XP. To run the application, they have to purchase old hardware, which is not ideal. Using the web and code libraries built for MIDI sequencing, I was able to create a modern solution that runs on current hardware.
