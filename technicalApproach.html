Technical Approach

Architecture Overview

The application is built with a Vue.js frontend, which is a JavaScript framework. To load and play MIDI files and handle notes played, the application uses Tone.js, which is a JavaScript library that can work with MIDI events and files. Vue applications run on the Node JavaScript runtime. This allows the application to run in the browser, which runs on the user's computer. If the user desires, they can plug in their MIDI keyboard to play notes that will be played in the browser.

Development Patterns

To manage this project, I followed the Agile Scrum methodology. Each development portion in the class, which lasted a few weeks, was the length of the next sprint. I had all of the tasks for the project tracked in Jira, and I tracked each sprint's progress using a burn-down chart, which I generated each week. This meant that I could make sure I was completing tasks on time and keeping on track. At the end of each development portion, I would generate a test plan, and test every feature that I developed. This was to ensure that the work I was completing was correct and accurate.
