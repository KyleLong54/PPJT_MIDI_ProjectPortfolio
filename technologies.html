Technologies

Technologies Used

Vue.js

This project was built using Vue.js. Vue was recommended to me by my professor, as it is a modern and popular JavaScript web development framework. I really enjoyed working in Vue, the way the code is organized, and its simplicity.

Tone.js

Tone.js is a JavaScript library that can translate MIDI events into tones. This means that, without any extra hardware, a web application that uses this library can natively play MIDI files. I was also recommended this library, this time by a graduated student that my professor got me in touch with.

JavaScript

JavaScript was used throughout the project, as it was created with JavaScript libraries and frameworks.

Node

Vue.js runs using Node.js. I used Node.js and the Node Package Manager during the development. These were used for running the application for testing and development, building the project to be pushed as a release, and installing the required packages for development.

Jira

To manage my workflow and to help track my tasks for each sprint, I used Jira. Each iteration of the project was treated as a sprint, and all tasks that needed to be completed were put into the sprint. Each week, a burndown chart was created, showing the pace of the work compared to the time left in the sprint.

Technologies Learned

Other than JavaScript, I had to learn each of the above technologies for this project. I had never worked in a JavaScript framework before starting this project, including Vue.js. Vue.js requires Node.js, which I had never used, so I learned the Node Package Manager and Node for this project. I had worked in Agile sprints before, but never performing any tracking with Jira. Also, as I had never created a MIDI application before, I had to learn Tone.js and how to work with MIDI files in the code.
